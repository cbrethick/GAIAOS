Module 01 PRIORITY BUILD

Flood Probability Forecasting

Predicts 0–100% flood inundation probability per district across India, 72 hours ahead. Uses rainfall + terrain + soil saturation signals fused together.

GOAL
72-hour flood risk map, district-level, 0–100% probability
MODEL
Phase 1: XGBoost (tabular) → Phase 2: ConvLSTM (spatial grids) → Phase 3: Physics-informed U-Net
INPUTS
7-day cumulative rainfall · soil moisture · elevation · river discharge · land use type · slope
LABELS
NDMA historical flood events · Sentinel-1 SAR inundation masks · IMD observed data
HARD PART
SAR preprocessing (speckle filter + terrain correction). Class imbalance — only 0.8% days have floods. Use focal loss.
DATASETS
ERA5 SMAP SRTM DEM Sentinel-1 SAR GloFAS Kerala 2018 CSV
TOOLS
xarray · rasterio · XGBoost · PyTorch · geopandas · SNAP (SAR)
BEAT
IMD operational forecast + naive persistence. Target: +0.05 AUC

Module 02 SWIN TRANSFORMER

Climate Pattern Forecasting

Predicts temperature anomalies, monsoon onset date, and drought index 30 days ahead at 25km grid resolution across India.

GOAL
30-day temperature anomaly map + monsoon onset ±5 days + drought index per district
MODEL
Swin Transformer on 3D ERA5 pressure-level grids. Seq2seq: past 30 days → next 30 days
INPUTS
ERA5: temperature, wind u/v, geopotential, specific humidity at 5 pressure levels, SST anomalies, sea ice
LABELS
ERA5 future states (self-supervised). IMD monsoon onset dates 1950–2024
HARD PART
Non-stationarity — 2024 climate ≠ 1990 climate. Time-weighted sampling + trend-aware normalization required
DATASETS
ERA5 (multi-level) NOAA SST CHIRPS rainfall IMD monsoon records
TOOLS
PyTorch · timm (Swin-T) · xarray · einops · Weights & Biases
OUTPUT
Animated 30-day anomaly maps, monsoon onset probability bell curve, SPI drought map

Module 03 TEMPORAL CNN

Crop Yield Prediction

Predicts district-level crop yield (wheat, rice, soybean) 6 weeks before harvest for 600+ Indian districts using satellite NDVI time series.

GOAL
Yield forecast per district, 6 weeks before harvest. Output: tonnes/hectare ± confidence interval
MODEL
Two-head ensemble: Temporal CNN on NDVI time series + XGBoost on tabular weather/soil. Late fusion with stacking.
INPUTS
MODIS NDVI weekly 500m, ERA5 rainfall + temperature, SMAP soil moisture, crop calendar, IMD monsoon onset
LABELS
ICRISAT district yield statistics 1966–2017. 600+ districts. Wheat, rice, sorghum, groundnut.
HARD PART
NDVI cloud noise in monsoon season. Implement Savitzky-Golay smoothing on NDVI before training. Temporal leakage prevention.
DATASETS
MODIS MOD13A1 SMAP Soil ICRISAT yields ERA5 FAO India
TOOLS
xarray · MODIS API · scikit-learn · PyTorch · scipy (SG filter) · geopandas
OUTPUT
District yield map with % deviation from 5-yr average. Color: green=surplus, red=deficit. Exportable CSV.
SAMPLE OUTPUT

Module 04 XGBOOST + CNN

Wildfire Risk Forecasting

7-day Fire Weather Index map at 1km resolution for India and Southeast Asia. Trained on 22 years of NASA FIRMS active fire detections.

GOAL
7-day animated fire risk grid at 1km. Top-20 highest-risk districts listed with cause breakdown
MODEL
XGBoost on weather + vegetation tabular features (fast, interpretable) + CNN on MODIS veg maps. Calibrated ensemble.
INPUTS
ERA5: wind speed, relative humidity, temperature, drought code. MODIS NDVI fuel load. SRTM slope. Past fire proximity.
LABELS
NASA FIRMS fire detections 2001–2023. Binary: fire/no-fire per 1km cell per day. 22 years of labels.
HARD PART
0.3% positive class (fires are rare). SMOTE-spatial oversampling + isotonic regression calibration for reliable probabilities.
DATASETS
NASA FIRMS ERA5 weather MODIS NDVI SRTM terrain Copernicus fuel type
TOOLS
XGBoost · scikit-learn · imbalanced-learn · rasterio · geopandas · folium
OUTPUT
Animated 7-day risk GIF + static probability map + feature importance chart per region

Module 05 GNN + U-NET

Traffic & City Growth Prediction

Predicts urban land expansion (5-year horizon) and peak traffic congestion corridors for 10 major Indian cities.

GOAL
5-yr urban growth probability map + daily congestion heat maps for Mumbai, Delhi, Bengaluru + 7 more cities
MODEL
U-Net semantic segmentation on Sentinel-2 time series (urban growth) + GNN on road intersection graph (traffic)
INPUTS
Sentinel-2 bands (B02-B12), OSM road graph topology, Meta HRSL population density, VIIRS night-lights radiance
LABELS
Sentinel-2 urban pixel change 2017→2022. OSM historical edit timestamps as proxy for growth areas.
HARD PART
Slum areas don't look like planned urban in satellite — use focal loss + augmentation. GNN requires OSMnx preprocessing for road graph.
DATASETS
Sentinel-2 L2A OpenStreetMap Meta HRSL VIIRS Night Lights GHSL Urban
TOOLS
PyTorch · segmentation-models-pytorch · OSMnx · PyTorch Geometric · sentinelsat · kepler.gl
OUTPUT
Urban expansion map overlay with crop module to show farmland loss. City congestion heatmap by hour of day.

Streamlit Starter Code
app.py · Streamlit · ~200 lines total
import streamlit as st
import folium, plotly.express as px
from streamlit_folium import st_folium
import pandas as pd, numpy as np

st.set_page_config(page_title="GaiaOS", layout="wide")

# Sidebar controls
module = st.sidebar.selectbox("Module", ["Flood", "Climate", "Crop", "Wildfire", "Traffic"])
date = st.sidebar.date_input("Forecast date")
thresh = st.sidebar.slider("Risk threshold", 0, 100, 50)

# Load predictions (your model output CSVs)
df = pd.read_csv(f"predictions/{module.lower()}_latest.csv")

# India risk map
m = folium.Map(location=[20.5, 78.9], zoom_start=5)
folium.Choropleth(geo_data="india_districts.geojson",
data=df, columns=["district", "risk"],
fill_color="RdYlGn_r", legend_name=f"{module} Risk %").add_to(m)
st_folium(m, width=800, height=500)

# Metrics row
col1, col2, col3 = st.columns(3)
col1.metric("High-risk districts", len(df[df.risk > thresh]))
col2.metric("Max risk", f"{df.risk.max():.0f}%")
col3.metric("Model AUC", "0.87")
